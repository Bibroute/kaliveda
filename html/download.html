Download

KaliVeda is available to download and build from source from our Github repository.


Source code

Source code is available from Github either in the form of compressed archives (tarballs) of released versions, or you can just clone/fork the repository in order to keep up to date with the latest developments and bugfixes using Git.

Releases
Source code for all released versions are available here. Releases can be downloaded as either a .zip or a .tar.gz archive.
Git
You can clone the git repository by doing:
$ git clone https://github.com/kaliveda-dev/kaliveda.git Cloning into 'kaliveda'... remote: Counting objects: 30161, done. remote: Total 30161 (delta 0), reused 0 (delta 0), pack-reused 30161 Receiving objects: 100% (30161/30161), 363.74 MiB | 9.01 MiB/s, done. Resolving deltas: 100% (21645/21645), done. Checking connectivity... done.
This will create a directory called 'kaliveda' containing the source code. To update the source code after the initial clone, you can do:
$ cd kaliveda $ git pull

Once you have downloaded the source tree, you need to Build & Install the toolkit.
